# Obsidian Regex Extractor

## Development

Install modules:
```bash
npm install
```

Build Javascript in Dev mode:
```bash
npm run dev
```

Build Javascript once:
```bash
npm run build
```

### Adjustments
- esbuild.config.mjs muss einen gültigen Pfad haben für den Entrypoint (wenn man das main.ts verschiebt, muss man es anpassen)
